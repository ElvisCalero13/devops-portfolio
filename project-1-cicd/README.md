# Project 1 - CI/CD Pipeline on AWS ECS

## Overview

This project demonstrates a complete CI/CD workflow for a containerized FastAPI application.

## Stack

- FastAPI
- Docker
- GitHub Actions
- Amazon ECR
- Amazon ECS Fargate
- Terraform

## Features

- Automated tests on push
- Docker image build
- Push to Amazon ECR
- Deployment to Amazon ECS
- Infrastructure as Code with Terraform

## Endpoints

- `/`
- `/health`
- `/version`

## Project Goals

- Demonstrate end-to-end CI/CD
- Show AWS deployment skills
- Keep the solution simple, cost-aware, and production-oriented
